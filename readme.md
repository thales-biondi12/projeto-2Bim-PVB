# 📊 Sistema de Cálculo Estatístico em C#

## 📌 Descrição do Projeto

Este projeto foi desenvolvido em C# utilizando WPF (Windows Presentation Foundation) com o objetivo de realizar cálculos estatísticos a partir de números inseridos pelo usuário.

O sistema permite inserir diversos valores numéricos em uma caixa de texto e calcular automaticamente:

- Quantidade de elementos
- Valor mínimo
- Valor máximo
- Média aritmética
- Variância amostral
- Desvio padrão amostral

O projeto foi desenvolvido como atividade acadêmica do curso técnico de informática.

# 🖥️ Interface do Sistema

O usuário deve inserir um número por linha na caixa de entrada do sistema.

### Exemplo:

```text
10
20
30
40
50
```

Após clicar no botão de cálculo, o sistema exibirá:

- Quantidade de elementos
- Menor valor
- Maior valor
- Média
- Variância
- Desvio padrão

---

# 🧮 Fórmulas Utilizadas

## Média Aritmética

```text
Média = Soma dos valores / Quantidade
```

## Variância Amostral

```text
Variância = Σ(x - média)² / (n - 1)
```

## Desvio Padrão

```text
Desvio Padrão = √Variância
```
